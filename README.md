Battleship
A console version of the classic Battleship game written in Python.
Play against the computer on a 6x6 grid and try to sink all enemy ships first.
Game Rules
The board size is 6x6.
Ships are placed randomly:
1 ship of 3 cells
2 ships of 2 cells
4 ships of 1 cell
The player and the computer take turns shooting by entering coordinates.
The first to destroy all opponent ships wins.
Controls
When it is your turn, enter coordinates in the format x,y
Example: 2,3
where x is the row number and y is the column number.
Features
Object-oriented design
Separate classes for game logic:
Dot - a point on the board
Ship - represents a ship
Board - represents the game board
Player, User, AI - player logic
Game - controls the game process
Random ship placement
Collision and boundary validation
Simple console interface
